### 1. In which OSI layer the Wi-Fi standard/protocol fits?
Wi-Fi (IEEE 802.11) operates in:
- **Physical Layer (Layer 1)** → Handles radio transmission, frequencies, modulation.
- **Data Link Layer (Layer 2)** → Handles MAC addressing, framing, error detection.

### 2. Wi-Fi device properties and generation mapping

Example (Laptop/Phone):
- Standard: 802.11ac / 802.11ax
- Frequency Bands: 2.4 GHz / 5 GHz
- Channel Width: 20/40/80/160 MHz
- Max Speed: Depends on standard
- MIMO Support: Yes
- Security: WPA2/WPA3

| Generation | Standard   | Frequency | Max Speed |
|------------|------------|----------|----------|
| Wi-Fi 4    | 802.11n    | 2.4/5 GHz | ~600 Mbps |
| Wi-Fi 5    | 802.11ac   | 5 GHz     | ~3.5 Gbps |
| Wi-Fi 6    | 802.11ax   | 2.4/5 GHz | ~9.6 Gbps |

### 3. What is BSS and ESS?

- **BSS (Basic Service Set)**:
  - A single Access Point with connected devices.
  - Identified by BSSID.

- **ESS (Extended Service Set)**:
  - Multiple APs connected together forming one network.
  - Same SSID across APs for seamless roaming.

---

### 4. Basic functionalities of Wi-Fi Access Point

- Provides wireless connectivity to devices
- Acts as a bridge between wireless and wired network
- Handles authentication and encryption
- Manages traffic and device association
- Assigns IP (if DHCP enabled)

---

### 5. Difference between Bridge Mode and Repeater Mode

| Feature        | Bridge Mode                     | Repeater Mode                  |
|---------------|-------------------------------|--------------------------------|
| Function      | Connects two networks         | Extends Wi-Fi coverage         |
| Connection    | Wired/Wireless                | Wireless only                  |
| Performance   | Better                        | Reduced due to retransmission  |
| Use Case      | Network expansion             | Coverage extension             |

---

### 6. Difference between 802.11a and 802.11b

| Feature     | 802.11a        | 802.11b        |
|-------------|----------------|----------------|
| Frequency   | 5 GHz          | 2.4 GHz        |
| Speed       | Up to 54 Mbps  | Up to 11 Mbps  |
| Range       | Shorter        | Longer         |
| Interference| Less           | More           |

---

### 7. 2.4 GHz vs 5 GHz observations

| Feature        | 2.4 GHz                  | 5 GHz                    |
|----------------|--------------------------|--------------------------|
| Range          | Longer                   | Shorter                  |
| Speed          | Lower                    | Higher                   |
| Interference   | High (Bluetooth, etc.)   | Low                      |
| Penetration    | Better through walls     | Poor                     |

---

### 8. Difference between IEEE and WFA

- **IEEE (Institute of Electrical and Electronics Engineers)**:
  - Develops Wi-Fi standards (e.g., 802.11)

- **WFA (Wi-Fi Alliance)**:
  - Certifies devices for interoperability
  - Defines Wi-Fi branding (Wi-Fi 4, 5, 6)

---

### 9. Wi-Fi Internet Connectivity Backhaul

Types of backhaul:
- Wired (Fiber, Ethernet)
- Wireless (Microwave, Point-to-Point)

Example:
- Home/College Backhaul: Fiber broadband (ISP)
- Properties:
  - High bandwidth
  - Low latency
  - Reliable connectivity

---

### 10. Wi-Fi Topologies and Use Cases

1. **Infrastructure Mode**
   - Devices connect via Access Point
   - Used in homes, offices

2. **Ad-Hoc Mode**
   - Device-to-device communication
   - Used in temporary networks

3. **Mesh Topology**
   - Multiple APs interconnected
   - Used in large campuses, smart cities

4. **Point-to-Point**
   - Direct link between two locations
   - Used for building-to-building connectivity
